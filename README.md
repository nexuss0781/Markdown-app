# Nexuss Markdown Viewer

## Project Description
The **Nexuss Markdown Viewer** is a sleek, web-based application designed to provide a beautiful and intuitive way to view and navigate through Markdown files directly in your browser. It allows users to select a folder containing Markdown files and then browse, read, and enjoy their content with syntax highlighting and a responsive design.

## Features
*   **Beautiful Rendering**: Clean, readable presentation of your Markdown content.
*   **Syntax Highlighting**: Code blocks within Markdown files are automatically highlighted for better readability across various programming languages.
*   **File Navigation**: Easily browse and switch between all Markdown files within your selected folder via a convenient sidebar.
*   **Responsive Design**: Optimized for viewing on desktop, tablet, and mobile devices.
*   **Dark Theme**: An elegant dark mode that is easy on the eyes, with a toggle option.
*   **Fullscreen Mode**: View your Markdown content in an immersive fullscreen experience.
*   **Drag and Drop**: Easily load folders by dragging and dropping them onto the application window.

## How to Use
1.  **Open the Application**: Open the `Markdown-viewer.html` file in your web browser.
2.  **Select a Folder**: Click the "Select Folder" button (folder icon in the sidebar or the larger button in the main content area).
3.  **Browse Markdown Files**: Choose a folder on your device that contains `.md` or `.markdown` files.
4.  **View Content**: The sidebar will populate with a list of Markdown files. Click on any file to view its beautifully rendered content in the main display area.
5.  **Toggle Theme**: Use the moon/sun icon button in the header to switch between dark and light themes.
6.  **Fullscreen**: Use the expand icon button to enter or exit fullscreen mode.

## Technologies Used
*   **Frontend**: HTML5, CSS3, JavaScript
*   **Markdown Parsing**: [Marked.js](https://marked.js.org/)
*   **Syntax Highlighting**: [Highlight.js](https://highlightjs.org/)
*   **Icons**: [Font Awesome](https://fontawesome.com/)
*   **Fonts**: [Google Fonts (Inter, Source Code Pro)](https://fonts.google.com/)

## Setup
This is a client-side web application. To use it, simply download the `Markdown-viewer.html` file and open it with any modern web browser. No server-side setup or installation is required.
